# Section 9: The Importance of Component Certification

Certification ensures that products meet aviation requirements.

## Mandatory Certifications:
- FAA 8130  
- EASA Form 1  
- AS9100 / AS9120  
- OEM traceability documents  

## Additional Checks:
- Shelf life for rubber, elastomers, and seals  
- Batch and lot number verification  
- Electrostatic discharge compliance
